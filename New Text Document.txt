## TASK 1: Buat SupervisorDashboard.jsx
File baru: src/pages/supervisor/SupervisorDashboard.jsx

Halaman ini adalah dashboard analytics lengkap untuk supervisor. Berisi SEMUA data dari seluruh divisi dalam satu halaman scroll. Desain harus premium dan informatif.

### Section A: Header
- Greeting: "Dashboard Supervisor" dengan icon BarChart3
- Subheading: "Overview seluruh aktivitas Komprestasi IPB"
- Style: hero-gradient (purple gradient) rounded-xl p-6, text-white

### Section B: Overall Stats (4 cards horizontal)
Fetch dari Supabase:
1. Total Anggota — SELECT count(*) FROM users
2. Total Prestasi — SELECT count(*) FROM prestasi
3. Total Event — SELECT count(*) FROM events WHERE visibility IN ('internal', 'eksternal')
4. Total Lomba Aktif — SELECT count(*) FROM lomba

Style: bg-white rounded-xl border border-zinc-200 p-5 text-center, icon gradient di atas, angka font-black text-3xl

### Section C: Leaderboard Prestasi + Leaderboard Keaktifan (2 kolom)

KIRI — Leaderboard Prestasi:
- Heading: "Leaderboard Prestasi"
- Fetch semua prestasi, hitung poin per user:
  Juara 1=100, Juara 2=85, Juara 3=70, Harapan 1=55, Harapan 2=40, Harapan 3=25, Honorable Mention=15
- Fetch users name by user_id
- Sort by total poin DESC, tampilkan TOP 10
- Tampilan: table/list — Rank, Nama, Poin
- Rank 1: bg-amber-50 border-amber-200 (gold)
- Rank 2: bg-zinc-50 border-zinc-200 (silver)
- Rank 3: bg-orange-50 border-orange-200 (bronze)
- Rank 4+: bg-white

KANAN — Leaderboard Keaktifan:
- Heading: "Leaderboard Keaktifan"
- Fetch: event_registrations WHERE attended = true AND score > 0
- Group by user_id, SUM(score)
- Fetch users name
- Sort DESC, TOP 10
- Tampilan sama seperti Leaderboard Prestasi

Style per leaderboard: bg-white rounded-xl border border-zinc-200 p-6
Layout: grid grid-cols-1 lg:grid-cols-2 gap-6

### Section D: Monthly Achievement Trends (full width)
- Heading: "Tren Prestasi Bulanan"
- Fetch semua prestasi, group by bulan dari tanggal_prestasi (kalau null fallback ke created_at)
- Hitung jumlah prestasi per bulan untuk 12 bulan terakhir
- Cek apakah recharts sudah di package.json. Kalau ada, pakai:
  import { LineChart, Line, XAxis, YAxis, CartesianGrid, Tooltip, ResponsiveContainer } from "recharts"
  Buat line chart: X=bulan, Y=jumlah, stroke="#6231D4"
- Kalau recharts tidak ada, buat SVG chart manual (polyline)
- Container: bg-white rounded-xl border border-zinc-200 p-6

### Section E: Distribusi Prestasi per Cabang (kiri) + Distribusi per Cluster (kanan)

KIRI — Per Cabang:
- Heading: "Distribusi Prestasi per Cabang"
- Fetch prestasi, group by cabang, count
- Tampilkan horizontal bar chart (bisa pakai recharts BarChart atau div bars manual)
- Gradient bars: from-royal-400 to-royal-600

KANAN — Per Cluster:
- Heading: "Distribusi per Cluster"
- 3 cards: Penulisan (count), Bisnis (count), Desain (count)
- Fetch prestasi, group by cluster, count
- Donut chart kalau bisa (recharts PieChart) atau 3 stat cards
- Warna: Penulisan=royal, Bisnis=amber, Desain=blue

Layout: grid grid-cols-1 lg:grid-cols-2 gap-6

### Section F: Event Overview (full width)
- Heading: "Event Terbaru"
- Fetch: SELECT * FROM events ORDER BY date DESC LIMIT 10
- Tampilkan table: Title, Date, Visibility (badge), Pendaftar count
- Fetch pendaftar count per event: SELECT event_id, count(*) FROM event_registrations GROUP BY event_id
- Style: bg-white rounded-xl border border-zinc-200 overflow-hidden
- Table header: bg-zinc-50
- Rows: hover:bg-zinc-50

### Section G: Aktivitas Terbaru (full width)
- Heading: "Aktivitas Terbaru"
